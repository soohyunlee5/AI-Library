# Contributing

Thanks for your interest in contributing! This project keeps things simple.

## Getting Started
- Fork the repository and create a topic branch from `main`.
- Install dependencies: `npm install`
- Run locally: `npm run dev`
- Lint before pushing: `npm run lint`

## Making Changes
- Keep PRs focused and small when possible.
- Write clear commit messages (Conventional Commits encouraged but not required).
- Don’t commit secrets or credentials.

## Pull Requests
- Ensure the app builds and basic flows work locally.
- Confirm lint passes and there are no type errors.
- Include a short description of the change and any screenshots if UI changes.

## Community & Conduct
- Be kind and respectful. We follow a common-sense code of conduct: be welcoming, assume good intent, no harassment.
- If something goes wrong, open an issue and we’ll sort it out together.

Happy hacking!
